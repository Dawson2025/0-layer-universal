# Legacy Content from Consolidated Sublayers

This directory contains all content that was migrated from the old separate sublayers (0.06 through 0.14) when they were consolidated into `sub_layer_0.05-0.014_setup`.

## Directory Structure

- **mcp_servers/** - All content from `sub_layer_0.10_mcp_servers_and_tools_setup`
  - Contains MCP server documentation, configuration guides, and setup instructions
  - Includes the existing MCP file tree structure
  - Core system documentation and troubleshooting guides

- **universal_protocols/** - All content from `sub_layer_0.13_universal_protocols`
  - Protocol documentation and standards
  - Includes: cli_recursion, file_documentation_and_organization, framework_orchestration
  - memory_handling, observability, protocol_writing_standard, and more

- **universal_tools/** - All content from `sub_layer_0.12_universal_tools`
  - Universal tools documentation and configurations

- **other_setup/** - Content from sublayers 0.06, 0.07, 0.08, 0.09, 0.11, 0.14
  - Coding app setup (0.06)
  - Environment setup (0.07)
  - Apps/browsers/extensions setup (0.08)
  - AI apps and tools setup (0.09)
  - Model setup (0.11)
  - Agent setup (0.14)

## Why This Content Is Here

When consolidating the 10 separate setup sublayers into a single unified structure, we needed to preserve all existing documentation while transitioning to the new hierarchical file tree navigation system.

This legacy content directory serves as:
1. A preservation of all existing documentation
2. A reference for the new file tree structure
3. A transitional resource during the consolidation process

## Relationship to the New File Tree

The new universal setup file tree (`0.01_universal_setup_file_tree_0/`) provides a hierarchical navigation system across all setup dimensions. This legacy content represents the detailed documentation that will eventually be reorganized and integrated into appropriate locations within that hierarchy.
