# Cursor MCP Setup

## Config location
- `~/.cursor/mcp.json`

## Known issues
- Cursor tool exposure can be inconsistent on Linux/WSL (server connects but tools may not show). See the `general-issues/` docs and the Cursor-specific writeups.

## Recommended pattern
- Use the universal wrapper scripts (generated by `automation/scripts/mcp_manager.py --scope user`) to stabilize PATH/env when launching MCP servers.
- Prefer Playwright MCP for browser automation; use alternatives only when Cursor tool exposure bugs force a workaround.
