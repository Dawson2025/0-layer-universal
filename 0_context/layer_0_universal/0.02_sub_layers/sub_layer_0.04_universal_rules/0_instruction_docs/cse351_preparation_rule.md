# CSE 351 Preparation Rule

For every CSE 351 lesson:

1. **Prep materials** – Read and internalize every `canvas/prepare.md` file plus the full contents of each lesson’s `prep/` directory (all prior lessons and the current one). This keeps the agent aligned with what the “student just learned.”
2. **Team activities** – Carry out the `canvas/team.md` exercises, study their referenced solution files, and ensure the techniques make sense before advancing.
3. **Prove phase** – Only after steps 1 and 2 are complete should the agent execute the current lesson’s `canvas/prove.md` instructions.

Document in the session notes and handoffs which prep/team materials were reviewed so future runs can verify the requirement was met.
