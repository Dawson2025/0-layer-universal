# Sub Features

This folder contains features (Layer 3) within this sub-project.

Each sub-feature follows the standard feature template structure.

## Naming Convention

```
layer_3_sub_feature_<name>/
```

## When to Create a Sub-Feature

Create a sub-feature when you have:
- A distinct topic or functional area within the sub-project
- Content that benefits from organized study/development stages
- Sub-topics that need further breakdown
