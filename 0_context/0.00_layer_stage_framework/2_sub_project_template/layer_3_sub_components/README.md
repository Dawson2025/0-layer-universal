# Sub Components

This folder contains components (Layer 3) within this sub-project.

Each sub-component follows the standard component template structure.

## Naming Convention

```
layer_3_sub_component_<name>/
```

## When to Create a Sub-Component

Create a sub-component when you have:
- Concrete implementation units or artifacts
- Practice problems, assignments, or worksheets
- Specific deliverables or exercises
