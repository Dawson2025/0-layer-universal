# Universal Layer Template (0.x)

Use this when defining the universal layer for any ecosystem.

## Slots
- 0.01_basic_prompts_throughout: session init, what-to-do-next, core prompts.
- 0.02_software_engineering_knowledge_system: general SE knowledge map.
- 0.03_universal_principles: philosophies, values.
- 0.04_universal_rules: hard constraints (git, terminal, security).
- 0.05_os_setup: OS-specific setup (macOS/Linux/Windows).
- 0.06_coding_app_setup: IDE/editor configuration.
- 0.07_apps_browsers_extensions_setup: general apps, browsers, extensions.
- 0.08_ai_apps_tools_setup: AI clients/CLIs and integrations.
- 0.09_ai_models: approved models and guidance.
- 0.10_universal_tools: cross-project scripts/utilities.
- 0.99_stages: stage folders and status template.

## Stages (0.99)
- 0.01 instructions
- 0.02 planning
- 0.03 design
- 0.04 development
- 0.05 testing
- 0.06 criticism
- 0.07 fixing
- 0.08 archives

Copy this template, rename for your universal context, and populate per slot.
