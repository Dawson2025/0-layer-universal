# LangTrak Dev Agent System

Research project for designing how AI agents collaborate to develop LangTrak.

## Purpose

Define the agent system for LangTrak development: which agents exist (mapped to the 10-layer feature hierarchy), what context each agent needs, how they delegate work, and what tools they use.

## Key Questions

1. What agent roles map to LangTrak's layers (L2 Infrastructure → L11 Orchestration)?
2. How do cross-cutting layers (Enhancements, Admin, Orchestration) affect delegation?
3. What MCP tools does each layer agent need?
4. How should context cascade through the dependency chain?
5. What does a concrete development workflow look like?
