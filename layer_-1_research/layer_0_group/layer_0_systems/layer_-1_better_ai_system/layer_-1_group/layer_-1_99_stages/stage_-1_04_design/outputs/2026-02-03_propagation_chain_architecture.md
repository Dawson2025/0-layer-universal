## Quick Summary

AI context flows through a 4-layer propagation chain:

1. **Knowledge** (.0agnostic/ 01-05) — source of truth for content
2. **Agnostic Source** (0AGNOSTIC.md) — hub/entry point referencing knowledge
3. **Tool Files** (CLAUDE.md, AGENTS.md, etc.) — generated by agnostic-sync.sh + .1merge overrides
4. **Agent Action** — runtime loading from tool files + on-demand reads
