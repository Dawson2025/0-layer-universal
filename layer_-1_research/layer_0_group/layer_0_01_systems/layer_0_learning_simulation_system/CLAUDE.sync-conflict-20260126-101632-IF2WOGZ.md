## Overview
Research project exploring the Multi-Scale Discovery Environment (MSDE) - a next-generation STEM learning platform that merges real physics-based simulation with interactive learning.

## Purpose
- Design a unified simulation environment where physics, automation, and learning converge
- Research three-layer simulation architecture (Macro/Physics/Micro)
- Explore NVIDIA technology stack (Genesis, Newton, ALCHEMI, Warp)
