# Context Inheritance

**Need**: [Nested Systems](../README.md)

---

- SHOULD enable child systems to inherit context from parent systems via context chain
- SHOULD allow child systems to override parent context locally
- MUST NOT require child systems to duplicate parent content
- SHOULD use the three-tier knowledge pattern for context flow between nesting levels
