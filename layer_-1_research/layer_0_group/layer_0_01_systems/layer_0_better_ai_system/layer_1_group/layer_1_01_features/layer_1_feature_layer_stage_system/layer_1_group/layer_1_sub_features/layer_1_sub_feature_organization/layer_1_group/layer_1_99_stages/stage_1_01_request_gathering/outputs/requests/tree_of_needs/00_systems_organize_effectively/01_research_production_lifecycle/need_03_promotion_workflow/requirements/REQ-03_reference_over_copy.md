# Reference Over Copy

**Need**: [Promotion Workflow](../README.md)

---

- SHOULD prefer referencing research content from production rather than copying it
- SHOULD use the three-tier knowledge pattern: pointer → distilled → full content
- MUST avoid duplicating large content between research and production
- MAY create distilled summaries in production that link to full research
