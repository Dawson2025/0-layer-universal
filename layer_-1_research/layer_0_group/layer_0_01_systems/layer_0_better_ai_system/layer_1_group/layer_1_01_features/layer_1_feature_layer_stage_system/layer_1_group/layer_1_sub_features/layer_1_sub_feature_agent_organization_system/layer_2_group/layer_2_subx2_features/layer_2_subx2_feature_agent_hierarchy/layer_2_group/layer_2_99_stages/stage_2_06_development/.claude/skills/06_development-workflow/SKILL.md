# 06_development Workflow Skill

<!-- section_id: "9eb3bec5-4e58-4a9f-8675-35956f0c80ac" -->
## When to Use
- When entering the 06_development stage
- When performing 06_development activities
- When completing 06_development deliverables

<!-- section_id: "3348344e-d863-4212-b4e5-2951af22de89" -->
## Workflow Steps
1. **Initialize**: Check hand_off_documents/ for incoming context
2. **Execute**: Perform 06_development activities
3. **Document**: Place outputs in outputs/
4. **Handoff**: Prepare context for next stage

- `hand_off_documents/` - Incoming/outgoing context
- `ai_agent_system/` - Agent configurations
