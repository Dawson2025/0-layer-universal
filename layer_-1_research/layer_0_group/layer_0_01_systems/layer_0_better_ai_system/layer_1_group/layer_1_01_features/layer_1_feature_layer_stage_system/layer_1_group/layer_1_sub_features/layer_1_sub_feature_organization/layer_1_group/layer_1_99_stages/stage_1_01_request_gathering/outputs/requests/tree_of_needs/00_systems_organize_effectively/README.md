# Systems Organize Effectively

**Root Need**: Every system has a structured home for research, production, and instantiations.

<!-- section_id: "4ed4b2fe-f152-4140-b3cf-1e8180fb1536" -->
## Vision

Any system — AI, education, software — needs three structural versions:
1. **Research** (experimental): Where new ideas are explored without risk to stability
2. **Production** (stable): Tried-and-true patterns that agents and users rely on by default
3. **Instantiation** (personalized): Per-user or per-context instances that inherit from production

## Key Insight

The research/production/instantiation (R/P/I) pattern is the organizational foundation. Without it, experimental changes contaminate stable systems, users can't get personalized experiences, and features have no clear lifecycle from idea to delivery.
