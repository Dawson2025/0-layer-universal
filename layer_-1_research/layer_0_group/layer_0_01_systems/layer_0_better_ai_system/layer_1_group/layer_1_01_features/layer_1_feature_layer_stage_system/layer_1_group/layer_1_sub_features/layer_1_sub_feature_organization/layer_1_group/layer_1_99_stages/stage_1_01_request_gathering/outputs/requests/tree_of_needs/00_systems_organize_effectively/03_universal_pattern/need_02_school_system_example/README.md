# Need: School System Example

**Branch**: [Universal Pattern](../README.md)

<!-- section_id: "7a771370-8787-48cf-8795-60258cb58399" -->
## Definition

The school system MUST serve as the concrete example demonstrating the R/P/I pattern, including knowledge graph awareness, student modeling, adaptive learning, prerequisite tracking, and career alignment.

<!-- section_id: "77a21d37-833f-42ae-8af3-7589a86efa3e" -->
## Why This Matters

Abstract patterns need concrete examples. The school system is the primary use case: the AI system researches better teaching methods, production contains validated pedagogical patterns, and each student gets a personalized learning experience.

<!-- section_id: "b16fc23a-b6d6-463b-a379-d942d18454e8" -->
## Acceptance Criteria

- [ ] School system architecture documented as an R/P/I example
- [ ] Research features identified (knowledge graph, student modeling, adaptive learning, etc.)
- [ ] Per-student instantiation structure defined
- [ ] Knowledge graph, student profile, and career goals mapped to entity context
