# Student Modeling

**Need**: [School System Example](../README.md)

---

- MUST create a student model that tracks what the student knows and doesn't know
- MUST update the model as the student learns (assessments, interactions, feedback)
- SHOULD identify misconceptions, not just missing knowledge
- SHOULD support confidence levels for each concept (not just binary know/don't-know)
