# Validation Before Promotion

**Need**: [Promotion Workflow](../README.md)

---

- MUST validate research findings before promoting to production
- MUST check that the finding has been tested (stage 07) or reviewed (stage 08)
- SHOULD require at least one concrete example or working implementation
- SHOULD NOT promote speculative or untested findings
