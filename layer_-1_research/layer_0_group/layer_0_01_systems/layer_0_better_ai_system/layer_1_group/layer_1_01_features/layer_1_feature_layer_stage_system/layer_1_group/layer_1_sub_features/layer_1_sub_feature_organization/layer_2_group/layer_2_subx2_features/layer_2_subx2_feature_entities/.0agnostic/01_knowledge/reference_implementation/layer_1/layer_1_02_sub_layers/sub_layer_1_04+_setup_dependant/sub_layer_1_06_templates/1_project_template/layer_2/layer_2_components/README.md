# Components

This folder contains components (Layer 2) within this project.

Each component follows the standard component template structure (`3_component_template`).

## Naming Convention

```
layer_2_component_<name>/
```

<!-- section_id: "501b977a-9d9a-495b-9026-bc1755ee675a" -->
## When to Create a Component

Create a component when you have:
- Concrete implementation units or artifacts
- Practice problems, assignments, or worksheets
- Specific deliverables or exercises

<!-- section_id: "1bcb019a-b4d3-44d0-92fc-2346c29c388c" -->
## Note on "Sub" Prefix

The "sub" prefix indicates nesting within the **same type**:
- Component inside a project = **component** (different types, NO "sub")
- Component inside a component = **sub_component** (same type nesting)
- Project inside a project = **sub_project**

Since this is a component inside a project (different types), it does NOT use the "sub" prefix.
