## Issues RESOLVED

<!-- section_id: "8abfa564-5455-4b24-bb4d-81f76281ca4a" -->
### 1. Ubuntu Login Loop - FIXED ✅
- **Problem:** Ubuntu GUI login loops back to login screen
- **Cause:** GDM3 + NVIDIA hybrid graphics issue ("Session never registered, failing")
- **Solution:** Switched from GDM3 to LightDM
  ```bash
  apt install lightdm
  echo '/usr/sbin/lightdm' > /etc/X11/default-display-manager
  systemctl disable gdm3
  reboot
  ```
- **Status:** RESOLVED - Ubuntu now boots and logs in normally

<!-- section_id: "e10a9c46-8e47-4e87-8882-3654b2dca33b" -->
### 2. TTY Access Not Working
- **Problem:** Ctrl+Alt+F1/F2/F3 not switching to text console
- **Possible causes:**
  - GNOME capturing the keys
  - Display manager configuration
  - Need to try F4, F5, or F6 instead

<!-- section_id: "b8ffb056-f3ae-4824-b78b-916c44430bc6" -->
### 3. Claude Root Restriction
- **Problem:** `claude --dangerously-skip-permissions` can't run as root
- **This is a security feature, not a bug**
- **Solution:** Create non-root user or run without that flag
