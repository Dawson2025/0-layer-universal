## Technical Notes

<!-- section_id: "8dc8c976-85e0-43da-a3a5-70bed1b99852" -->
### IPv6 vs IPv4
The connection established over IPv6 (`[2a01:4f8:1c1a:885b::1]:22000`), which is preferred over IPv4 for direct connections. This provides better performance and avoids NAT traversal issues.

<!-- section_id: "618b14ae-73f1-4994-8f55-7763459a9a8a" -->
### API-Based Configuration
Using the Syncthing REST API instead of manual GUI configuration allows for:
- Automated, reproducible setup
- No need to manually click through GUI dialogs
- Easier troubleshooting via JSON responses
- Can be scripted for future installations

<!-- section_id: "0ccb42e4-29a2-46f7-ab9a-c4e8dbd57246" -->
### Connection Security
All Syncthing connections use TLS 1.3 with device certificates for mutual authentication. Data is encrypted in transit even over untrusted networks.
