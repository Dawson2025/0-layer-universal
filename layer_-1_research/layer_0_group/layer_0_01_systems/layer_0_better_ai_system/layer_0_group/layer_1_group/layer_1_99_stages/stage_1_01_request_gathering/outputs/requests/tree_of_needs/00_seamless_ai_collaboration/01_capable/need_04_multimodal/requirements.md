## Capability-Specific Concerns

From the 01_capable perspective, the key questions are:

- **Voice Input**: Does the system support speech-to-text? (Vibe Typer, Whisper, etc.)
- **Voice Output**: Does the system support text-to-speech?
- **Visual Output**: Can the system generate diagrams, charts, images?
- **Integration**: Do these modalities work with the rest of the system?

These are capability questions - whether the system *can* do something.

The 05_engaging branch focuses on whether these capabilities create an *enjoyable* experience.
