# Components

This folder contains components (Layer 3) within this sub-project.

Each component follows the standard component template structure (`3_component_template`).

## Naming Convention

```
layer_3_component_<name>/
```

## When to Create a Component

Create a component when you have:
- Concrete implementation units or artifacts
- Practice problems, assignments, or worksheets
- Specific deliverables or exercises

## Note on "Sub" Prefix

The "sub" prefix indicates nesting within the **same type**:
- Component inside a project = **component**
- Component inside a component = **subcomponent**
- Project inside a project = **subproject**

Since this is a component inside a sub-project (different types), it's just called a "component", not a "subcomponent".
