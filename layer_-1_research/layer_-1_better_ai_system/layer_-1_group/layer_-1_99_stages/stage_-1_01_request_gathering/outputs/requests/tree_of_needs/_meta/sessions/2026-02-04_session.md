## New Branch Added: 06_context_flow

**Question**: Does context reach agents?

**Why This Branch**: AI agents suffer from "amnesia" (no memory across sessions) and may not receive the right context at the right time. This branch addresses the need for context to flow properly.

### Needs in This Branch

| Need | Description |
|------|-------------|
| **need_01_no_amnesia** | Agents remember/recover previous session context |
| **need_02_context_propagation_works** | Context flows through hierarchy correctly |
| **need_03_static_context_available** | Rules, knowledge, principles accessible |
| **need_04_dynamic_context_available** | State, history, handoffs accessible |
| **need_05_entry_points_right_detail** | Entry files have enough to direct, not overwhelm |
| **need_06_navigation_to_deeper_details** | Agent can find more info when needed |
