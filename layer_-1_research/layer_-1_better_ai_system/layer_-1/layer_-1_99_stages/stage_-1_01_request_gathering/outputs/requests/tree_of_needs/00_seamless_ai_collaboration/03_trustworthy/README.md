## Core Question

> "Can I trust AI to behave correctly?"

---

## Description

For AI to be trustworthy, it must:
1. **Follow rules** - Respect defined constraints and guidelines
2. **Be predictable** - Behave consistently, no surprises
3. **Stay bounded** - Respect scope, don't overstep authority

## Key Insight

Trust comes from **constraints** and **consistency**. AI that follows rules, behaves predictably, and respects boundaries can be relied upon.

The solution is:
- Clear rule hierarchies (what rules apply where)
- Conflict resolution (what happens when rules conflict)
- Scope definitions (what AI can and cannot do)
- Consistent behavior patterns (same input → same behavior)
