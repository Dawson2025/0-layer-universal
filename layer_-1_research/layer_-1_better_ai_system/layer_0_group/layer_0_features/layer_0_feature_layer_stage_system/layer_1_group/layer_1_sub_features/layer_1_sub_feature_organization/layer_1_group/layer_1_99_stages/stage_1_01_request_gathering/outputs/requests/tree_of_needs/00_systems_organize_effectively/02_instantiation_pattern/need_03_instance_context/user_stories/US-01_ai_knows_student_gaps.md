# AI Knows Student's Gaps

**As an** AI agent,
**I want to** access the student's knowledge state from their instance context,
**So that** I can identify gaps and tailor explanations to what they don't yet understand.

## Acceptance Criteria

- [ ] Knowledge state is accessible from instance entity
- [ ] Gaps are identifiable (topics marked as unknown or weak)
- [ ] AI can reference the knowledge graph to understand gap dependencies
