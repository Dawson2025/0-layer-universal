# Gemini Config Component

## Purpose
Defines patterns for Gemini CLI configuration.

## Files at Entity Root
- `GEMINI.md` - Primary configuration file

## GEMINI.md Pattern
```markdown
# [Entity Name] Gemini Configuration

## Context
[Entity description]

## Instructions
- [Instruction 1]
- [Instruction 2]
- [Instruction 3]

## Key Files
- [Important files]

## Navigation
- [Parent/child entities]
```

## Integration
Gemini CLI reads `GEMINI.md` for context configuration.
