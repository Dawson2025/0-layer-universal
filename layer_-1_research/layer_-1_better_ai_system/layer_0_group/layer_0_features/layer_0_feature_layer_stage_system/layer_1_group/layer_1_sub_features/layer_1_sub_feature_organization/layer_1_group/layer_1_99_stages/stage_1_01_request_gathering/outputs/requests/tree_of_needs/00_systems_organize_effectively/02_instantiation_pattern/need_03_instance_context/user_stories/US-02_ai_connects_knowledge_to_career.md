# AI Connects Knowledge to Career

**As an** AI agent,
**I want to** connect the student's learning to their career goals,
**So that** I can prioritize topics and motivate the student with relevant applications.

## Acceptance Criteria

- [ ] Career goals are stored in instance context
- [ ] AI can map current coursework to career requirements
- [ ] AI can explain "why this matters" in terms of career relevance
