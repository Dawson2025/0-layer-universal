# Specific Nesting Pattern

## Nesting Levels
1. **OS**: wsl, linux_ubuntu, macos, windows
2. **Environment**: local, cloud/{aws,gcp,azure}
3. **Coding App**: cursor_ide, vscode, jetbrains, rstudio, terminal
4. **AI App**: claude_code_cli, codex_cli, gemini_cli, cursor_agent

## Path Example
```
specific/os/wsl/environment/local/coding_app/terminal/ai_app/claude_code_cli/config.md
```

## Resolution
Navigate down through levels to find most specific config.
