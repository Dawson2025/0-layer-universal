# School Has Own R/P/I Cycle

**As a** school system,
**I want** my own research/production/instantiation cycle,
**So that** educational features are developed, stabilized, and personalized independently.

## Acceptance Criteria

- [ ] School system has research entities for educational features
- [ ] School system has production patterns for validated teaching methods
- [ ] School system has per-student instances with personalized context
- [ ] School system follows the same R/P/I pattern as its parent
