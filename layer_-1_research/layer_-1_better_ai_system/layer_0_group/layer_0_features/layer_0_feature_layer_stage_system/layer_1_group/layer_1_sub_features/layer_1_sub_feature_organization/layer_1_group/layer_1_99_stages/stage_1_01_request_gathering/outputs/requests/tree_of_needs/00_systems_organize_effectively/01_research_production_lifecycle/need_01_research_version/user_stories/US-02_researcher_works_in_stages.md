# Researcher Works in Stages

**As a** researcher,
**I want to** work through stages (01-11) within a research entity,
**So that** my exploratory work follows a structured process from requirements to deliverables.

## Acceptance Criteria

- [ ] Can gather requirements in stage 01
- [ ] Can conduct research in stage 02
- [ ] Can design solutions in stage 04
- [ ] Can produce deliverables in stage 10
- [ ] Each stage has its own 0AGNOSTIC.md with methodology
