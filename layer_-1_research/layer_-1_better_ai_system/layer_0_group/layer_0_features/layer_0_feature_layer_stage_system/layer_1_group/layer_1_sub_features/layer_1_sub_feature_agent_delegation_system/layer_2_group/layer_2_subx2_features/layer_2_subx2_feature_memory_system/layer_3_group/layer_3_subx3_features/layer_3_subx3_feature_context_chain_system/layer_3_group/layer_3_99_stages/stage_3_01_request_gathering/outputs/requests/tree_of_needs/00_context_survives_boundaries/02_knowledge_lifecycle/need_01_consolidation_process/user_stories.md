# Need: Consolidation Process — User Stories

---

### US-1: Developer distills research at stage boundary
**As the** developer finishing the research stage,
**I want** a clear protocol for distilling 19 research files into ~5 knowledge files,
**So that** future agents get competence from 260 lines instead of 5,000.

**Acceptance**: Protocol exists, followed once successfully, compression ratio documented.

---

### US-2: Agent triggers consolidation reminder
**As an** agent transitioning from one stage to another,
**I want** to be reminded that consolidation should happen,
**So that** knowledge files are updated before the next stage begins.

**Acceptance**: Stage-workflow skill includes consolidation step at transitions.

---

### US-3: Developer verifies consolidation quality
**As the** developer reviewing a consolidation,
**I want** to check that knowledge files reference their sources and are genuinely distilled (not copied),
**So that** the three-tier pattern is maintained.

**Acceptance**: Checklist exists. Every knowledge file passes it.
