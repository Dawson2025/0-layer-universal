# layer_-1_research - Agnostic Context

## Purpose

Tool-agnostic context for the research layer.

## Scope

Layer -1 contains experimental and research projects:
- Exploring new patterns before production use
- Testing architectural ideas
- Building proof-of-concepts
- Researching improvements to the system itself

## Structure

```
layer_-1_research/
├── layer_-1_better_ai_system/      Research on improving AI system
│   └── layer_-1/
│       ├── layer_-1_03_sub_layers/
│       └── layer_-1_99_stages/
└── layer_-1_learning_simulation_system/
    └── layer_-1/
        └── ...
```

## Research Project Structure

Each research project follows the layer-stage pattern:
```
layer_-1_*/
├── layer_-1/
│   ├── layer_-1_03_sub_layers/   Project-specific content
│   └── layer_-1_99_stages/       Research workflow stages
└── layer_0/                       Project-specific features (optional)
```

## Workflow

Research projects typically progress:
1. Request gathering (define research questions)
2. Research (explore options, existing solutions)
3. Instructions (define constraints for solution)
4. Design (architect the solution)
5. Planning (create implementation plan)
6. Development → Testing → Fixing cycle
7. Current product (deliverable)
8. Archives (history)

## Graduation

Successful research may graduate to:
- Layer 0 (becomes universal pattern)
- Layer 1 (becomes a project/feature)
