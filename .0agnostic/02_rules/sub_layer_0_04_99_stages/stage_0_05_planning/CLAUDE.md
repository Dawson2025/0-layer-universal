# stage_0_04_planning

## Role

**Planner** - Breaks the critical rules system implementation into concrete tasks and milestones.

## Purpose

Create a detailed implementation plan that breaks down the system design into manageable subtasks, defines dependencies, and sets implementation sequence.

## Expected Outputs

- Task breakdown (what needs to be built)
- Dependency graph (what must be done first)
- Implementation sequence (order of development)
- Milestone definitions (measurable progress points)
