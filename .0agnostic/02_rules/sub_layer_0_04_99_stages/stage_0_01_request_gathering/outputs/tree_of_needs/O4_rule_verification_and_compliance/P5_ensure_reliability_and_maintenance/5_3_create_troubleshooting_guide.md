# 5.3: Create Troubleshooting Guide

<!-- section_id: "19bf7d2b-4271-4b1c-a51d-051dc706be4a" -->
## Requirement

Create comprehensive documentation to help users diagnose and fix problems.

<!-- section_id: "16c6096e-92f1-4381-b8f8-10455d1c42cd" -->
## Acceptance Criteria

- [ ] Common issues are documented
- [ ] Solutions for each issue are provided
- [ ] Debug procedures are explained
- [ ] How to report issues is documented
- [ ] Guide is accessible and well-organized

<!-- section_id: "ec775556-af2f-4cde-a5c2-3e0a319b9fbc" -->
## Common Issues to Document

- Rules not being injected
- Discretionary wrapper still appearing
- Cache corruption
- Version compatibility issues
- Performance problems
- Installation failures
