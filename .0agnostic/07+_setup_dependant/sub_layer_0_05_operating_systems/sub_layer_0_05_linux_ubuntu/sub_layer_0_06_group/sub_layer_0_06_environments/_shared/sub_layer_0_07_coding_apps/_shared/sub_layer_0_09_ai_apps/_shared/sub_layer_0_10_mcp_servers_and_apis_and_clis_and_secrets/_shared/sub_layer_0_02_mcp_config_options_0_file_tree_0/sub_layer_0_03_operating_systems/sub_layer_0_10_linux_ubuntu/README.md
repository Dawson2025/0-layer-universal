# Linux (Ubuntu) MCP Notes

<!-- section_id: "81789aef-b886-4f6a-9360-c99cc8f40762" -->
## Scope
Use this when running AI tools on Ubuntu (native Linux, not WSL).

<!-- section_id: "38faaa07-7069-4d7c-b04c-82e2f5afcb0b" -->
## Key prerequisites
- Node.js available in PATH (or via NVM).
- Playwright browsers installed and `PLAYWRIGHT_BROWSERS_PATH` set if needed.

<!-- section_id: "e5f8cbfc-b816-46f9-87a4-b0338a7a2cd0" -->
## Notes
- Prefer Playwright MCP for browser automation.
- If you don’t need a visible browser, headless mode is usually more reliable.

<!-- section_id: "77692736-b0e7-4518-acec-b417c76c61f6" -->

See `0.04_ai_apps/` for per-app MCP setup instructions on Ubuntu.
