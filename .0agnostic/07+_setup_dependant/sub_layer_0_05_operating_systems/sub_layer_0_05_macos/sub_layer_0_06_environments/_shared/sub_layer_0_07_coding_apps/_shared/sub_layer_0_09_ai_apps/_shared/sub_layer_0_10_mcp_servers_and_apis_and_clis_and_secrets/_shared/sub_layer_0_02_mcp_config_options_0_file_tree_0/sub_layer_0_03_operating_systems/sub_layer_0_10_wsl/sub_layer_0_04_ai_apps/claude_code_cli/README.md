# Claude Code CLI MCP Setup (WSL)

<!-- section_id: "0e2b604e-c691-4d2b-a716-ad39bfcc3c00" -->
## Config location
- `~/.claude.json` (stores MCP server configs per project)
- `claude mcp ...` CLI commands can add/remove servers

<!-- section_id: "1bbaf64e-2ff2-4306-9d17-68479c7672cd" -->
## Recommended setup
- Prefer universal wrapper scripts generated by:
  - `automation/scripts/mcp_manager.py --scope user`

<!-- section_id: "4899b3ef-92eb-4803-bca3-177f2bd126bf" -->
## WSLg headed Playwright
- Use Playwright MCP via `--config` JSON (see `mcp_servers/playwright-mcp/INSTALL.md`).
- Ensure the MCP server process has WSLg runtime env:
  - `DISPLAY=:0`
  - `WAYLAND_DISPLAY=wayland-0`
  - `XDG_RUNTIME_DIR=/mnt/wslg/runtime-dir`
