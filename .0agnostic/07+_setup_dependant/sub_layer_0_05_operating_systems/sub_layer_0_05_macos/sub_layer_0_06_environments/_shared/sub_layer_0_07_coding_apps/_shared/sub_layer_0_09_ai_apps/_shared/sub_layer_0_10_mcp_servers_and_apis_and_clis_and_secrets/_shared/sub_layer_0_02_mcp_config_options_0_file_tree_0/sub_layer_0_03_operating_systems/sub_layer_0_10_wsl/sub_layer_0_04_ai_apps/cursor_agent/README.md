# Cursor MCP Setup (WSL)

<!-- section_id: "67bee0c5-33f8-44a2-bf19-384f2a7fffc5" -->
## Config location
- `~/.cursor/mcp.json`

<!-- section_id: "ce64fcea-6a29-42ef-85de-d7ab0a494cb3" -->
## Notes (WSL)
- Cursor may connect to MCP servers but not expose tools reliably (tool exposure limitations/bugs can apply).
- Prefer using Gemini/Codex for browser automation on WSL when Cursor tool exposure blocks browser MCP usage.

<!-- section_id: "abba706a-f573-4854-91eb-81da9256e0ab" -->
## Recommended setup
- Use wrapper scripts generated by `automation/scripts/mcp_manager.py --scope user` to stabilize PATH/env.
