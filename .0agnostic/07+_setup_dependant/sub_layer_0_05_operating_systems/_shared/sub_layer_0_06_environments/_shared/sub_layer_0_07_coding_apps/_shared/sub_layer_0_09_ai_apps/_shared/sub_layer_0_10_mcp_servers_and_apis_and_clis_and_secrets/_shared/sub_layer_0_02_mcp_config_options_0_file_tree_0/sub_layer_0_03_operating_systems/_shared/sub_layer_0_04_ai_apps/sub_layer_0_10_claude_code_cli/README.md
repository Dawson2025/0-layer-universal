# Claude Code CLI MCP Setup

<!-- section_id: "32b1aec7-f186-43eb-a104-aaa3ff703b7a" -->
## Config location
- `~/.claude.json` (stores MCP server configs per project, plus other Claude settings)
- CLI commands exist for managing MCP servers (e.g., `claude mcp add ...`)

<!-- section_id: "cee231d9-de68-4891-a69c-1050740ec659" -->
## Recommended pattern
- Prefer the universal wrapper scripts generated by `../../../../../../0.06_automation/scripts/mcp_manager.py --scope user`
- For Playwright on WSLg, use a `--config` JSON approach (see `0.05_mcp_servers/playwright-mcp/INSTALL.md`)

## Notes
- If you add MCP servers “per project” via Claude’s CLI, document the choice in the relevant project layer handoff docs.
