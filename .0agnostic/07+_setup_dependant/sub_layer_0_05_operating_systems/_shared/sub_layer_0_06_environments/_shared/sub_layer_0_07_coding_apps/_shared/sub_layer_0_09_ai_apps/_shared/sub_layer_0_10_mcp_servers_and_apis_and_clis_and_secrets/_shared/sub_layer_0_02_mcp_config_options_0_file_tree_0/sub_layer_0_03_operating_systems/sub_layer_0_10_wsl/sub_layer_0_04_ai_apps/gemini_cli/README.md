# Gemini CLI MCP Setup (WSL)

## Config location
- `~/.gemini/settings.json` (single canonical file; contains `mcpServers`)

## Recommended setup
- Prefer universal wrapper scripts generated by:
  - `automation/scripts/mcp_manager.py --scope user`
- Gemini typically points to wrappers under `~/.config/mcp/servers/` (e.g., `mcp-playwright-generic.sh`).

## WSLg headed Playwright
- Ensure WSLg env vars exist for headed rendering:
  - `DISPLAY=:0`
  - `WAYLAND_DISPLAY=wayland-0`
  - `XDG_RUNTIME_DIR=/mnt/wslg/runtime-dir`
- Prefer Playwright MCP `--config` JSON to pass WSLg stability args:
  - `--ozone-platform=wayland`
  - `--enable-features=UseOzonePlatform`
