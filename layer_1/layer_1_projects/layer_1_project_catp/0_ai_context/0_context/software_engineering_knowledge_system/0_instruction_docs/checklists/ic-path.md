# Individual Contributor Learning Path
*From Novice to Senior IC*

<!-- section_id: "e83eb7bb-52b6-423e-ae0e-a754fef69423" -->
## Purpose

This checklist tracks your progression through the essential knowledge areas for individual contributors in software engineering, from entry-level through senior IC roles.

<!-- section_id: "13a43630-9c65-4ed8-87f0-9e9446dfcd42" -->
## How to Use This Checklist

For each topic, mark your current mastery level:

- `[ ]` Not started
- `[N]` Novice - Aware of concepts, can explain basics
- `[C]` Competent - Can implement with guidance
- `[P]` Proficient - Can implement independently
- `[E]` Expert - Deep understanding, can teach
- `[M]` Master - Industry-recognized expertise

Also track your experience:
- `рџ“љ` Learned (studied/read about it)
- `рџ’»` Built (practical hands-on experience)
- `рџ‘ЁвЂЌрџЏ«` Taught (can teach/mentor others)

## Progress Tracking

<!-- section_id: "3577a7e8-2b09-4c41-a2f4-1f1a69cf29aa" -->
### Junior IC (0-2 years)
**Target**: Complete Phase 1 (Fundamentals) and Phase 2 (Implementation)
- [ ] Can write functional code in at least one language
- [ ] Understands version control
- [ ] Can write basic tests
- [ ] Familiar with one frontend OR backend framework
- [ ] Understands basic data structures & algorithms

<!-- section_id: "77558d63-d9a0-41a2-bdea-d6575489f136" -->
### Mid-Level IC (2-5 years)
**Target**: Master Phase 2, Complete Phase 3 (Professional Development)
- [ ] Proficient in at least one full stack or specialized area
- [ ] Writes comprehensive tests
- [ ] Participates in code reviews effectively
- [ ] Understands CI/CD and deployment
- [ ] Can design and implement features independently
- [ ] Comfortable with databases and APIs

<!-- section_id: "ee63ec56-f51c-42f1-9999-73fd850cd97c" -->
### Senior IC (5+ years)
**Target**: Complete Phase 4 (Senior IC Skills)
- [ ] Designs scalable systems
- [ ] Expert in primary tech stack
- [ ] Mentors junior/mid engineers
- [ ] Contributes to technical decisions
- [ ] Identifies and addresses technical debt
- [ ] On-call and incident response experience
- [ ] Can lead small projects
