# Sub-Components

This folder contains sub-components (Layer 3) within this feature.

Each sub-component follows the standard component template structure (`3_component_template`).

## Naming Convention

```
layer_3_sub_component_<name>/
```

## When to Create a Sub-Component

Create a sub-component when you have:
- Concrete implementation units or artifacts within the feature
- Practice problems, assignments, or worksheets
- Specific deliverables or exercises

## Note on "Sub" Prefix

The "sub" prefix indicates nesting within a **feature context**:
- Component inside a project = **component**
- Component inside a feature = **sub_component** (in feature context)
- Component inside a component = **sub_component** (same type nesting)

Since this is a component inside a feature context, it uses the "sub" prefix.
