# Handoff Creation Skill

## When to Use
- At the end of a work session
- When transitioning to a different agent
- When context needs to be preserved

## Handoff Document Contents
1. Current status summary
2. Work completed this session
3. Pending tasks
4. Decisions made
5. Open questions
6. Next steps

## Steps
1. Review session activity
2. Identify key accomplishments
3. Note any blockers or issues
4. Create structured handoff document
5. Save to appropriate location
