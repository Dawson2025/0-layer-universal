## Key Decisions

| Decision | Rationale |
|----------|-----------|
| 0AGNOSTIC.md as source of truth | Single place to edit, generates tool-specific files |
| .0agnostic/ for detailed resources | Keeps lean context separate from detailed rules |

| Session files with timestamps | Enables chronological tracking |
| Index.md for quick lookup | Agents can quickly find recent context |
