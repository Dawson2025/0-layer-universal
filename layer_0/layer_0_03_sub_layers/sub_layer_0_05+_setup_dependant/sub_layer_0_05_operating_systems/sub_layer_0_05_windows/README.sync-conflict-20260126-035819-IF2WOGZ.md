# Windows Setup

Setup documentation specific to Windows systems.

## Windows-Specific Considerations

### Package Management
- Winget and Chocolatey for system packages
- Microsoft Store for GUI applications

### Permissions
- Administrator privileges for system-wide installs
- Execution policy for PowerShell scripts
