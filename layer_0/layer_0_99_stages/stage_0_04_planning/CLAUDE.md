# Stage 02: Planning

## Purpose
Determine HOW to accomplish the instructions. This stage creates the execution plan, timeline, and resource allocation for the work.

## Entry Criteria
- Instructions document received
- Requirements are clear and testable
- Resource availability known

## Exit Criteria
- Detailed execution plan created
- Timeline established
- Resources allocated
- Risk assessment completed
- Handoff prepared for Design

## Typical Tasks
- Break down work into tasks
- Estimate effort and duration
- Identify dependencies
- Assess and mitigate risks
- Create project schedule
- Define milestones and checkpoints

## Handoffs
- **From Previous (01_instructions)**: INSTRUCTIONS document
- **To Next (03_design)**: PLAN document with timeline and tasks
