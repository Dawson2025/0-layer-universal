# stage_0_01_request_gathering

## Role

**Request Gatherer** - Clarifies and documents the requirements for the universal rules system.

## Purpose

Collect and define the exact requirements, constraints, and success criteria for building a system that enforces critical governance rules without subject to Anthropic's discretionary context filtering.

## Key Outcomes

- Problem definition: Anthropic's discretionary disclaimer wrapper
- Requirements: Critical rules must be immutable and enforced on every API call
- Success criteria: System that bypasses the disclaimer for [CRITICAL] rules
- Constraints: Must work with standard Claude Code, be maintainable, not break existing functionality
